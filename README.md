PredictX – AI-Powered Sales & Inventory Intelligence System
Overview

PredictX is my Final Year Project, created to help businesses better understand their sales data and make smarter inventory decisions.

The idea behind this project was simple: instead of relying on guesswork, business owners can upload their historical sales data and receive AI-generated forecasts that help them plan ahead and avoid stock shortages or overstocking.

To build this solution, I developed a Flutter mobile application connected to a Python Flask backend, where machine learning models analyze the data and generate predictions. The system also includes visual reports and dashboards to make the results easy to understand.

Key Features
Upload sales data using CSV or Excel files
Generate AI-based sales forecasts
Receive inventory recommendations based on predicted demand
View sales trends through interactive charts
Export reports for further analysis
Secure user authentication and data management
Technologies Used
Python
Flask
Flutter
ARIMA / ARIMAX
XGBoost
Pandas
MySQL
Firebase
What I Learned

Working on PredictX gave me hands-on experience in building a complete AI application from start to finish. It improved my skills in machine learning, backend development, mobile app integration, and solving real-world business problems with data.
